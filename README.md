## 🤖 AI Automation Workflow

The "AI News" section is completely autonomous. 
1. `scripts/update_news.py` runs daily via GitHub Actions.
2. It fetches top AI trends from external sources.
3. It generates `data/ai-news.js`, bypassing CORS and ensuring instant client-side loading.
4. The frontend renders dynamic cards with AI-generated placeholders.

## 🚀 Development & Setup

### Local Preview
Since the project uses **ES Modules** for Three.js, it requires a local server:

1. **Python (Recommended):**
   ```bash
   python -m http.server
   ```
2. **VS Code:** Use the **Live Server** extension.

### News Update Script
To manually refresh the AI news feed:
```bash
pip install -r requirements.txt
python scripts/update_news.py
```
